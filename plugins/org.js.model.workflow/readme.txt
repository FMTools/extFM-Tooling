this project allows us to extend jwt meta model.  

1. project structure:
src/main/model: extended meta model
src/gen/java: generated java codes
src/main/java: extended menus, buttons, actions and useful help methods
conf: configuration model and configuration view

2. jwt installation
eclipse -> help -> install new software -> juno -> soa development -> java workflow tooling
